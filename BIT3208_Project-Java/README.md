# Week 9: Java Servlets, Session Management, and Cookies

## Student Login System

A complete Student Login System built with Java Servlets and JSP demonstrating session management, cookies, and "Remember Me" functionality.

## Technologies Used
- Java 8+
- Java Servlets 4.0
- JSP 2.3
- Apache Tomcat 9
- HTML5, CSS3

## Features Implemented
- ✅ User Login
- ✅ Session Management (HttpSession)
- ✅ Remember Me (Cookies)
- ✅ Session Information Display
- ✅ Logout Functionality
- ✅ Role-Based Access Control
- ✅ Error Handling
